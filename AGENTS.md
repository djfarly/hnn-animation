# HNN Animation

Manim Community project for neural network explainer videos. Uses the [manim-neural-network](https://github.com/JPM2002/manim-neural-network) plugin for NN visuals.

## Docs

- **Manim Community**: https://docs.manim.community/en/stable/installation.html
- **manim-neural-network**: https://github.com/JPM2002/manim-neural-network

## Run

```bash
source .venv/bin/activate
manim -pql scene_nn_demo.py NNDemo
```
